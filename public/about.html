The Stack
what it’s built on · end to end
▷ Backend / ML
  • Python 3.11 · Flask · Flask-CORS
  • scikit-learn (LogReg, Random Forest, Gradient Boosting)
  • pandas · numpy for feature engineering
  • Hosted on Vercel serverless Python
  • Trained on 6,582 UFC fights (2001–2024)
▷ Frontend / Deploy
  • Vanilla HTML / CSS / JS — no framework
  • Archivo Black · Space Grotesk · Space Mono
  • Static assets over Vercel CDN
  • GitHub Actions for CI
  • Lighthouse 100 / 100 / 100 / 100
