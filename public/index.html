Abstract.

tl;dr · what we built · what it costs
§ text

We present an ensemble classifier for predicting Ultimate Fighting Championship bout outcomes. The system combines five independently trained models — XGBoost, LightGBM, logistic regression, random forest, and support vector machine — using equal-weight averaging with Platt calibration. The training corpus comprises 6,582 fights spanning 2001–2024, with 86 engineered features per fighter pair including striking efficiency, grappling rates, recent form momentum, strength of schedule, and physical attributes. All rate statistics use 5-year exponentially decayed rolling averages (λ = 0.3). On a strictly held-out test set of 1,284 fights (2022–2024), the ensemble achieves 68.4% classification accuracy, a Brier score of 0.218, log loss of 0.614, and AUC-ROC of 0.724. Each prediction includes 95% bootstrap confidence intervals and SHAP-based feature attribution. We report all results transparently, including calibration analysis, failure modes, and known limitations.

§ 1.1   results

We evaluate performance on a strictly held-out test set of 1,284 fights from January 2022 through December 2024. No information from this period was used during feature engineering, hyperparameter tuning, or training. We report four complementary metrics, following Brier (1950) and Gneiting & Raftery (2007) for probabilistic forecast evaluation.1

Calibration. A well-calibrated model produces predicted probabilities that match observed win rates. The 70–80% confidence bucket shows near-perfect calibration at 71.2%. The model exhibits slight overconfidence above 90%, consistent with limited sample sizes at extreme levels (n=86).

Methodology.

86 features · 5 categories · λ=0.3 exponential decay
§ 2.1   features

The feature pipeline produces 86 features per fighter pair across five categories: striking efficiency (21), grappling rates (14), physical attributes (8), experience and momentum (18), and opponent quality (25). All rate statistics use exponentially decayed rolling averages, weighting recent fights more heavily.2

I — Architecture
  • XGBoost — η=0.1, depth=6, 500 rounds
  • LightGBM — lr=0.05, 31 leaves
  • Logistic Regression — L2, C=1.0
  • Random Forest — 500 trees, depth=12
  • SVM — RBF kernel, C=10, γ=auto
w(t) = e−λt,   λ = 0.3
P(win) = (1/5) Σi Pi(win | x)
II — Known Failure Modes
  • No injury or training camp data available
  • Weight cut quality is unobservable
  • Stylistic matchup nuance only partially captured
  • Small sample size at extreme confidence (>90%)
  • Does not model in-fight adjustments
  • No accounting for judging tendencies

Probability calibration uses Platt scaling on a validation fold. Feature attribution uses SHAP values from the ensemble mean. The full methodology breakdown walks through the pipeline end to end.
